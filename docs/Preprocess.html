Documentation for the Preprocessing script

Create indices for the smoother Hi-C data viewer.

usage: preprocess.py [-h] [-v VERBOSITY] {init,repl,norm,grid-seq-norm} ...

Positional Arguments

cmd

Possible choices: init, repl, norm, grid-seq-norm

Sub-command that shall be executed.

Named Arguments

-v, --verbosity

@todo make this do sth

Sub-commands:

init

Create a new index.

preprocess.py init [-h] [-d DIVIDEND] [-a PATH] index_prefix chr_len

Positional Arguments

index_prefix

Path where the index shall be saved. Note: a folder with multiple files will be created.

chr_len

Path to a file that contains the length (in nucleotides) of all chromosomes. The file shall contain 2 tab seperated columns columns: The chromosome names and their size in nucleotides. The order of chromosomes in this files will be used as the display order in the viewer.

Named Arguments

-d, --dividend

Divide all coordinates by this number. Larger numbers will reduce the index size and preprocessing time. However, bins with a size below this given number cannot be displayed.

-a, --annotations

Path to a file that contains annotations for the genome. File should be in the gff3 fromat.

repl

Add a replicate to a given index.

preprocess.py repl [-h] [-g {a,b,both,neither}] index_prefix path name

Positional Arguments

index_prefix

Prefix that was used to create the index (see the init subcommand).

path

Path to the .pre1.bed file that contains the aligned reads.

name

Named Arguments

-g, --group

Possible choices: a, b, both, neither

(default: “neither”)
