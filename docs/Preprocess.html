Documentation for the Preprocessing script

Create indices for the smoother Hi-C data viewer.

usage: preprocess.py [-h] [-v VERBOSITY]
                     {init,repl,norm,grid-seq-norm,ddd-sample,ddd-load} ...

Positional Arguments

cmd

Possible choices: init, repl, norm, grid-seq-norm, ddd-sample, ddd-load

Sub-command that shall be executed.

Named Arguments

-v, --verbosity

@todo make this do sth

Sub-commands:

init

Create a new index.

preprocess.py init [-h] [-d DIVIDEND] [-a PATH] index_prefix chr_len

Positional Arguments

index_prefix

Path where the index shall be saved. Note: a folder with multiple files will be created.

chr_len

Path to a file that contains the length (in nucleotides) of all chromosomes. The file shall contain 2 tab seperated columns columns: The chromosome names and their size in nucleotides. The order of chromosomes in this files will be used as the display order in the viewer.

Named Arguments

-d, --dividend

Divide all coordinates by this number. Larger numbers will reduce the index size and preprocessing time. However, bins with a size below this given number cannot be displayed.

-a, --annotations

Path to a file that contains annotations for the genome. File should be in the gff3 fromat.

repl

Add a replicate to a given index.

preprocess.py repl [-h] [-g {a,b,both,neither}] index_prefix path name

Positional Arguments

index_prefix

Prefix that was used to create the index (see the init subcommand).

path

Path to the file that contains the aligned reads.

name

Name for the new replicate.

Named Arguments

-g, --group

Possible choices: a, b, both, neither

Which analysis group to place the new replicate in when opening the interface. (default: “neither”)

norm

Add a normalization track to an index, using external sequencing data.

preprocess.py norm [-h] [-g {row,col,both,neither}] index_prefix path name

Positional Arguments

index_prefix

Prefix that was used to create the index (see the init subcommand).

path

Path to the file that contains the aligned reads.

name

Name for the new normalization track.

Named Arguments

-g, --group

Possible choices: row, col, both, neither

Where to to place the new normalization track when opening the interface. (default: “neither”)

grid-seq-norm

Add a normalization track to an index, using the datasets already in the index. This follows the approach used in the GRID-seq paper. This function does nothing if neither -v -a or -n is given. We suggest using -v first to visualize what is then added to the index.

preprocess.py grid-seq-norm [-h] [-d [VAL [VAL ...]]] [-m VAL] [-b VAL]
                            [-R VAL] [-D VAL] [-A VAL] [-v] [-a] [-n]
                            index_prefix name

Positional Arguments

index_prefix

Prefix that was used to create the index (see the init subcommand).

name

Name for the new normalization track.

Named Arguments

-d, --datasets

Datasets to include in the normalization, given by their Id’s use the <list> command to find the ids. (default: all datasets)

-m, --mapping_q

Consider only reads with this mapping quality or higher. (default: 0)

-b, --bin_size

Use this bin size. (default: 1000)

-R, --filter_rna

Only keep rows with an average RNA density of VAL reads per kb. (default: 100)

-D, --filter_dna

Only keep columns with a maximal DNA density of VAL reads per kb. (default: 10)

-A, --annotation

name of the annotation to use as bins for the RNA or ‘bins’ to use –bin_size sized bins over the full genome.

-v, --visualize

Visualize the RNA and DNA densities as well as the picked filters.

-a, --add_annotation

Turn the picked genes into an annotation track.

-n, --add_normalization_track

Add the picked genes as an normalization track.

ddd-sample

Sample distance dependant decay from a replicate input file.

preprocess.py ddd-sample [-h] in_path out_path

Positional Arguments

in_path

Path to the file that contains the aligned reads.

out_path

Where to store the distance dependant decay file output.

ddd-load

Load a distance dependant decay file into the index.

preprocess.py ddd-load [-h] index_prefix in_path chr_list [chr_list ...]

Positional Arguments

index_prefix

Prefix that was used to create the index (see the init subcommand).

in_path

Path to the file that contains distance dependant decay.

chr_list

List of chromosomes which to normalize with this distance dependant decay.
